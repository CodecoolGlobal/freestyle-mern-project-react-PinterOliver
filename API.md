# Login

#### Request
``` Javascript
fetch('api/login', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    userName: String,
    password: String,
  }),
})
```
#### Response if succesful
``` Javascript
{token: String}
```
